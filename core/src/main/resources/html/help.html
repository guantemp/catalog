快速上手

欢迎使用一体化工作台！左侧菜单栏集中了所有核心模块，点击即可切换。

  • 桌面 — 数据概览与快捷入口
  • 品牌 / 分类 / 商品 — 基础数据维护
  • 文件上传 — 批量导入图片与资料
  • 零售 / 采购 / 库存 — 业务核心操作（开发中）

常用功能

🔍 搜索菜单

侧边栏顶部搜索框，输入关键词快速定位菜单项。

📌 折叠侧边栏

点击底部「折叠」按钮，为工作区腾出更多空间。

📂 多标签管理

点击菜单项自动新建标签，支持切换与关闭。

👤 用户中心

右上角头像下拉菜单，可修改设置或退出登录。

通知提醒

右上角铃铛图标显示未读通知数量，点击可查看详情。

💡 提示： 帮助页面内容完全独立，不占用主页面加载资源，点击才加载。
